# Challenge 12: Redirect Output

## 🧩 Task
Write output to a file.

---

<details>
<summary>💡 Click to view solution</summary>

### 🔹 Redirect output using echo
```bash
echo "Hello Linux" > file.txt
```

### 🔹 Redirect output of a command (`ls`)
```bash
ls > file.txt
```

**Example output inside `file.txt`:**
```text
file1.txt
file2.log
mydir
script.sh
```
